This app is designed to give the animated movie Insides character "Anxiety" a profile where users can learn a little about the character and put them in their chair to calm them.
The key screen content view displays anxiety's profile picture and provides a option to "Show more details". The Sheetview displays another "Anxiety Profile" View
The User Interface features a vibrant orange theme and a destinctive character icon. A prominant orange card features a bio with the text"Hi my name is Anxiety! I worry about everything past and future and I'm working on building a relationship with joy". "
visual Elements: Below thr bio,there is a image placeholder and the text "Put Anxiety in her chair".
Development Environmemt: Xcode.
Target device: optimized  for modern iOS devices like the iphone 17 Pro
